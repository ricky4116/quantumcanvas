# 🌌 퀸텀 캔버스 (QuantumCanvas)
"모두가 함께 그려가는 무한한 세계"
사용자들이 자신만의 세계관을 설정하고, AI의 도움을 받아 확장하며, 타인과 협업 및 연재를 통해 수익화까지 가능한 공동 창작 플랫폼입니다.

## 📖 1. 프로젝트 개요
<p1>기획 배경: 개인의 상상력을 체계적인 세계관으로 구축하기 어렵다는 점과, 창작물에 대한 피드백 및 수익 창출의 장을 마련하기 위해 기획되었습니다.</p1>
사용자 가치: 창작 진입 장벽 완화(AI 지원), 공동 창작을 통한 스토리 확장, 유료 연재를 통한 창작자 수익 모델 제공.

🚀 2. 세부 기능 상세



1. 인증 및 사용자 관리



 스플래시 화면: 웹 접속 시 행성 궤도 애니메이션과 함께 로고가 서서히 나타나는 로딩 화면

 일반 로그인/회원가입: 이메일 기반 가입 및 DB 연동

 소셜 로그인 API: 구글, 카카오, 네이버 계정 연동을 통한 빠른 접근

 내 정보 및 계정: 프로필 관리 및 유료 연재 수익 정산을 위한 계좌 연동 기능

2. 지능형 세계관 구축 시스템 (핵심)



 7대 설정 탭 UI 구성: 창작자가 직접 기입하는 구조화된 데이터 필드

🌍 세계 개요: 세계관의 명칭, 장르, 전체 시놉시스 및 핵심 키워드

🪐 지리 / 우주: 대륙, 행성, 주요 도시의 물리적 환경 및 위치 정보

👤 인물: 등장인물의 이름, 종족, 성격, 계급 및 인물 간 관계도(Relationship Map)

📜 역사 / 연표: 주요 사건의 연도별 기록 및 타임라인 시각화 UI

🧪 기술 / 마법: 해당 세계의 과학 기술력, 초능력, 마법의 법칙 및 제약 사항

🛡️ 정치 / 사회: 국가 체제, 법률, 사회적 계급 구조 및 권력 기관

🎭 문화 / 종교: 예술 양식, 일상 관습, 종교적 신념 및 민속 신화

 인터랙티브 하이퍼링크: 텍스트 입력 시 @ 혹은 특정 키워드를 인식하여 다른 설정 페이지로 바로 연결되는 위키(Wiki) 방식의 참조 기능

 슬롯 관리: 1인당 3개 세계관 무료 생성, 추가 시 슬롯당 500원 결제 시스템

3. 창작 보조 AI 가이드 (AI Assistance - 스토리 대필 제외)



 설정 모순 검사: "이 캐릭터가 현재 지리적 위치에서 1시간 만에 저 도시로 갈 수 있나요?"와 같은 설정 오류 검증

 브레인스토밍 파트너: "이 국가와 적대적인 국가의 명칭이나 특징 후보를 5개만 추천해줘"와 같은 단순 아이디어 나열

 설정 요약 및 시각화: 방대한 텍스트 설정을 한눈에 보기 좋게 표나 리스트 형식으로 정리해 주는 기능

 부적절 내용 필터링: 창작 과정에서 부적절한 윤리적 질문이나 유해 콘텐츠 요청 시 답변 거부

4. 연재 및 수익화 시스템



 스토리 연재 에디터: 설정 탭을 옆에 띄워두고 참고하며 글을 쓸 수 있는 분할 뷰(Split View) 제공

 단계별 유료화: 1~5화는 무료 공개, 6화부터는 창작자가 유료/무료 여부를 선택하여 수익 창출

 결제 API 연동: 유료 에피소드 열람 및 세계관 슬롯 구매를 위한 결제 모듈

5. 커뮤니티 및 협업 시스템



 그리드 기반 탐색: 참여자 수, 추천 수 기준 세계관 나열 (리사이클러뷰 형식의 무한 스크롤)

 유튜브형 댓글 인터페이스: 대댓글(Thread), 작성자 고정, 수정/삭제 알림 기능

 참여형 토론 및 반영: 독자가 세계관의 특정 설정에 대해 '보완 의견' 댓글 작성 승인 프로세스: 원작자가 댓글 중 좋은 아이디어를 '수락'하면 공식 설정 문서에 해당 내용이 추가/반영되는 기능
